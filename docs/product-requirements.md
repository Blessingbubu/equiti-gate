# Equiti Gate Product Requirements Document (PRD)

## Product Name

Equiti Gate

## Tagline

Building Wealth Through Real Estate

---

# Product Overview

Equiti Gate is a global real estate ecosystem that enables individuals to invest in professionally managed real estate projects, monitor their investments, and participate in long-term wealth creation.

The platform will expand over time to include property investment, Airbnb management, property sales, rentals, and property management services.

---

# Target Users

- First-time investors
- Experienced investors
- Property owners
- Property developers
- Airbnb hosts
- Real estate agents

---

# User Roles

## Visitor

Can:
- Browse the website
- View properties
- Learn about Equiti Gate
- Create an account

---

## Investor

Can:
- Complete profile
- Verify identity
- Deposit funds
- Invest
- View portfolio
- Track earnings
- Request withdrawals
- Watch reward advertisements
- Receive notifications

---

## Property Owner

Can:
- Submit a property
- Track approval
- View performance

---

## Administrator

Can:
- Manage users
- Manage properties
- Manage investments
- Approve withdrawals
- Review reports
- Publish announcements
