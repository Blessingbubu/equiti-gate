# Equiti Gate MVP Feature List

## Overview

The MVP represents the first public version of Equiti Gate.

The goal is to create a functional platform where users can:

- Learn about Equiti Gate
- Create accounts
- Explore properties
- View investment opportunities
- Manage their investor profile
- Track simulated investment activity

The MVP will focus on building trust, user experience, and platform foundations.

---

# MVP Goal

Create the first working version of Equiti Gate that demonstrates the complete user journey.

User journey:

Visitor
↓
Create Account
↓
Access Dashboard
↓
View Properties
↓
Select Investment Opportunity
↓
Track Investment

# MVP Success Criteria

The MVP is successful when:

- Users can understand the platform
- Users can create accounts
- Users can view investment opportunities
- Users can track their activity
- The platform foundation is ready for expansion
