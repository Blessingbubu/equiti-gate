# Equiti Gate User Journeys

## Overview

This document describes how different users interact with the Equiti Gate ecosystem.

---

# Journey 1: Visitor Becomes Investor

## Step 1: Landing on Website

A visitor arrives at Equiti Gate.

They can view:

- Company vision
- Available properties
- Investment opportunities
- How the platform works
- Frequently asked questions

---

## Step 2: Create Account

The user provides:

- Full name
- Email address
- Phone number
- Country
- Password

The system creates an account.

---

## Step 3: Account Verification

The user verifies:

- Email
- Phone number

Future versions may include identity verification.

---

## Step 4: Explore Investment Opportunities

The user views:

- Available investment plans
- Available properties
- Expected performance information
- Project details

---

## Step 5: Choose Investment

The user selects an investment option.

The system displays:

- Investment amount
- Project allocation
- Expected timeline
- Investment status

---

## Step 6: Complete Payment

Payment options may include:

- Cryptocurrency
- Mobile money
- Bank transfer

The transaction is recorded.

---

## Step 7: Track Investment

The investor dashboard displays:

- Active investments
- Portfolio value
- Project updates
- Earnings history
- Transactions

---

# Journey 2: Investor Withdraws Funds

## Step 1

Investor opens withdrawal section.

## Step 2

Investor selects:

- Amount
- Withdrawal method

## Step 3

System checks:

- Account status
- Available balance
- Security requirements

## Step 4

Withdrawal request is processed.

## Step 5

Investor receives notification.

---

# Journey 3: User Earns Rewards

## Step 1

Investor opens Rewards section.

## Step 2

User watches available advertisements.

## Step 3

System verifies completion.

## Step 4

Reward is added to user wallet.

## Step 5

User can view reward history.

---

# Journey 4: Property Owner Adds Property

## Step 1

Owner creates property submission.

Information required:

- Property name
- Location
- Images
- Description
- Property documents

---

## Step 2

Administrator reviews submission.

Possible outcomes:

- Approved
- Rejected
- Requires more information

---

## Step 3

Approved properties become visible on the platform.

---

# Journey 5: Administrator Management

Administrator can:

- View users
- Manage investments
- Review transactions
- Manage properties
- Publish updates
- Monitor platform activity

---

# User Experience Principles

Equiti Gate should provide:

- Simple navigation
- Clear information
- Transparency
- Fast response times
- Secure transactions
- Professional design
