# Equiti Gate Application Structure

## Overview

This document defines the structure, pages, navigation, and user interface sections of the Equiti Gate platform.

The platform consists of:

- Public website
- Investor portal
- Property management system
- Administration system

---

# 1. Public Website

The public website introduces Equiti Gate to visitors and potential investors.

---

## Homepage

Purpose:

Introduce Equiti Gate and explain the platform.

Sections:

### Hero Section

Contains:

- Company message
- Call-to-action buttons
- Real estate visuals

Example:

"Building Wealth Through Real Estate"

Buttons:

- Start Investing
- Explore Properties

---

## Statistics Section

Displays:

- Number of investors
- Properties funded
- Countries reached
- Total projects

---

## How It Works

Steps:

1. Create an account
2. Select an opportunity
3. Invest
4. Track performance
5. Receive distributions

---

## Investment Plans

Displays available plans:

- Foundation
- Growth
- Legacy

Each card shows:

- Minimum investment
- Benefits
- Features

---

## Featured Properties

Shows:

- Property image
- Location
- Type
- Investment status

---

## Trust Section

Displays:

- Security information
- Transparency features
- Company information

---

## Footer

Contains:

- About
- Contact
- Terms
- Privacy Policy
- Social links

---

# 2. Investor Portal

Accessible after login.

---

## Dashboard

Main overview page.

Displays:

- Total invested amount
- Portfolio value
- Active investments
- Rewards balance
- Recent activity

---

## Investment Page

Allows users to:

- View opportunities
- Select investment plans
- Review investments

---

## Portfolio Page

Displays:

- Current investments
- Property allocation
- Performance tracking
- Investment history

---

## Wallet Page

Features:

- Deposit funds
- Withdraw funds
- View balances
- Transaction history

---

## Rewards Page

Features:

- Available advertisements
- Reward balance
- Reward history
- Referral information

---

## Profile Page

Features:

- Personal information
- Security settings
- Verification status

---

# 3. Property System

## Property Listing Page

Displays:

- All available properties
- Search filters
- Location filters

---

## Property Details Page

Contains:

- Images

- Location
- Investment information
- Project updates

# 4. Administration System

## Admin Dashboard

Displays:

- Platform statistics
- User numbers
- Investment activity
- Property performance

---

## User Management

Allows administrators to:

- View users
- Verify users
- Manage accounts

---

## Property Management

Allows:

- Add properties
- Edit properties
- Upload reports
- Update project status

---

## Financial Management

Allows:

- Review transactions
- Manage withdrawals
- View reports
