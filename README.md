# Equiti Gate

## Building Wealth Through Real Estate

Equiti Gate is a real estate investment platform prototype that allows people worldwide to participate in property investment opportunities.

## Vision

To make real estate investment accessible to everyone by allowing people to start with small amounts and participate in global property projects.

## Platform Features

### Investors
- Create accounts
- Choose investment plans
- Track investments
- Monitor rewards
- View property projects

### Investment Plans

Bronze
- Minimum: $15

Silver
- Minimum: $50

Gold
- Minimum: $150

### Rewards System

Users can earn additional rewards through platform activities.
